Este exercício foi feito para um teste.

Para rodar o projeto, chamar por linha de comando o arquivo java nasa_test.java localizado dentro da pasta src e informar opcionalmente como parametro o valor do SOL (medição de dia em marte) cuja temperatura média deseja ver. 
Se não for informado o SOL, o resultado será a média da temperatura da última semana.
Se o SOL não for encontrado, retornará 0.0.
Em caso de problema com a conexão, retornará um erro.

Alternativamente, foi feito uma pagina index.html que funciona de maneira similar através de ajax, basta abrir o arquivo no seu browser.
